### 🔴 PROBLEMA ATUAL
```
✗ 98% dos clientes saem sem deixar contato
✗ Sem follow-up = sem conversão
✗ Falta dados sobre clientes
✗ Crescimento estagnado (~2-3 novos/mês)
```

### 🟢 SOLUÇÃO PROPOSTA
```
✓ QR Code na mesa → Formulário rápido
```

### 📈 RESULTADO

```
Mensagens personalizadas em 24h, 3d, 7d
```
