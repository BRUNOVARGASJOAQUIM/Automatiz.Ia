# 📊 GUIA: Como Usar a Proposta Comercial

---

## 🎯 Visão Geral

Você tem um **template profissional pronto** para usar com qualquer cliente.

Arquivo: `PROPOSTA-COMERCIAL-TEMPLATE.md`

---

## 🔧 Como Personalizar

### Passo 1: Substituir Dados Genéricos

Abra o arquivo e substitua:

```markdown
[NOME DO CLIENTE]
  ↓ Substitua por: "Sr. Altino Bar" ou nome real do cliente

[SEU NOME/EMPRESA]
  ↓ Substitua por: "Seu Nome" ou "Sua Empresa"

[seu-email@gmail.com]
  ↓ Substitua por: seu email real

[11 9XXXX-XXXX]
  ↓ Substitua por: seu WhatsApp real

[seu-site.com]
  ↓ Substitua por: seu site
```

### Passo 2: Ajustar Preços (Conforme Cliente)

Se é **Cliente Pequeno** (Bar):
```markdown
IMPLEMENTAÇÃO: R$ 2.500
MANUTENÇÃO: R$ 500/mês
```

Se é **Cliente Médio** (Rede 3-5 unidades):
```markdown
IMPLEMENTAÇÃO: R$ 3.500
MANUTENÇÃO: R$ 800/mês
```

Se é **Cliente Grande** (Franquia):
```markdown
IMPLEMENTAÇÃO: R$ 5.000+
MANUTENÇÃO: R$ 1.200/mês+
```

### Passo 3: Customizar ROI

Baseado no negócio do cliente:

```markdown
CLIENTE: Bar/Restaurante
  Leads atuais: 10/dia
  Taxa conversão: 15%
  Ticket médio: R$ 200-400

COM SISTEMA:
  Leads: 30-50/dia (+300%)
  Taxa: 22-25%
  Faturamento extra: R$ 25-50k/mês
```

## 💻 CONVERTER PARA PDF (Grátis)

### Google Docs

```
1. Copy-paste markdown no Google Docs
2. Formatar bonito (fonts, cores)
3. File → Download → PDF
```

### Microsoft Word

```
1. Copy-paste markdown em Word
2. Formatar (estilos)
3. File → Export as PDF
```

### Online (Não recomendo)

```
https://md-to-pdf.herokuapp.com/
Simplesmente colar markdown e fazer download
```
