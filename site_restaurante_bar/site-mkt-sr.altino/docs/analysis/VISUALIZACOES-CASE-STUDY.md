## 1. Fluxo de Conversão: ANTES vs DEPOIS

### ANTES (Sem Automação)
```
300 clientes/semana
    ↓
5-15 contatos capturados (2-5%)
    ↓
0-10% follow-up realizado
    ↓
<1% conversão final
    ↓
= 2-3 novos clientes/mês
= R$ ~0 em receita incremental
```

### DEPOIS (Com WhatsApp + N8N)
```
300 clientes/semana
    ↓
45-75 leads capturados (15-25%)
    ↓
80-95% follow-up automático
    ↓
5-8% conversão
    ↓
= 14-20 novos clientes/mês
= R$ 4.050/mês em receita bruta
= R$ 2.633/mês em lucro
= +650% crescimento
```

## 11. Riscos & Mitigações

### Risco 1: Baixa Adoção de QR Code

```
Risco: Apenas 2% dos clientes escaneiam
Impacto: -50% na receita projetada
Mitigação:
  • Posicionar QR em 3-5 locais visíveis
  • Incentivar com desconto (R$ 5 off)
  • Treinamento de staff para promover
  • Teste: Verificar adoção após 2 semanas
Probabilidade: BAIXA (20%) com mitigação
```

### Risco 2: Churn Acima do Esperado

```
Risco: Clientes não retornam após 1ª visita
Impacto: Taxa conversão cai para 2-3%
Mitigação:
  • Primeira mensagem muito importante
  • Oferta especial na 2ª visita
  • Reavaliar ambiente/produto
  • Follow-up em 3-4 dias
Probabilidade: MÉDIA (40%) sem otimização
```

### Risco 3: Qualidade Pobre de Dados

```
Risco: Leads preenchem dados incorretos
Impacto: Impossível contatar
Mitigação:
  • Validação em tempo real
  • Confirmação de WhatsApp
  • Campo obrigatório de número
  • Test follow-up imediato
Probabilidade: BAIXA (15%) com validação
```
