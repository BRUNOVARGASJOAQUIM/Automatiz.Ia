```
│        n8n webhook ↓                         │
│        Sheety (BD) ↓                         │

│          1️⃣ Cliente: Confirmação              │
│          2️⃣ Proprietário: Notificação         │

└───────────────────────────────────────────────┘

┌─ Campanhas (Envio em Massa) ─────────────────┐
│ Input: Título, Tipo, Conteúdo                │
│        ↓ Seleciona todos leads ↓             │
│        n8n webhook ↓                         │
│        Throttle: 5 msg/sec ↓                 │
│        Loop: Envia cada lead ↓               │
│        Sheety: Log de delivery               │
│        ✅ "Enviado para 150 leads"            │
└───────────────────────────────────────────────┘

┌─ Sistema de Fila (Offline) ──────────────────┐
│ ❌ Offline? Preenche form                     │
│    ↓ Salva em localStorage ↓                 │
│ ✅ Voltou online?                             │
│    ↓ Retry automático ↓                      │
│    Webhook enviado, stats atualizados ✨      │
└───────────────────────────────────────────────┘

┌─ Retry com Backoff ──────────────────────────┐
│ Tentativa 1: Falha ⏳ 1 seg                  │
│ Tentativa 2: Falha ⏳ 2 seg                  │
│ Tentativa 3: Falha ⏳ Fila                   │
│ Usuário vê: "Fila: 1 msg aguardando..."      │
└───────────────────────────────────────────────┘
```
