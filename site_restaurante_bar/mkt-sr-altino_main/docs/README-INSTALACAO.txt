SHOWZAP SR ALTINO - INSTALAÇÃO

1) Vá em Aparência > Temas > Adicionar tema > Enviar tema
2) Envie este ZIP
3) Ative o tema

4) Vá em Páginas > Adicionar nova
   Título: Área do Proprietário
   Slug: area-do-proprietario
   Modelo: Área do Proprietário
   Publicar

5) Crie um usuário administrador em Usuários > Adicionar novo

COMO O LOGIN FUNCIONA
- O botão "Área do proprietário" leva ao login normal do WordPress
- Depois do login, redireciona para /area-do-proprietario/
- Se não estiver logado, essa página nunca abre

LOGIN SOCIAL
- O WordPress não traz login social nativo
- Para isso, instale um plugin como Nextend Social Login
