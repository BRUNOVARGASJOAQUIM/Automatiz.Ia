# Automatiz.IA - Showzap Sr. Altino Theme

## 🚀 Plataforma de Automação WhatsApp Business

Sistema integrado de automação para gerenciamento de leads, vendas e atendimento via WhatsApp e n8n.

### 📁 Arquivos Principais

- **index.html** - Dashboard principal
- **PROPOSTA-SR-ALTINO.html** - Proposta comercial personalizada
- **ANALISE-PRESENCA-ONLINE.html** - Análise profissional de presença digital
- **proprietario.html** - Área do proprietário
- **login.html** - Autenticação

### 🔗 Rotas Disponíveis

- `/` - Dashboard
- `/proposta` - Proposta Sr. Altino
- `/analise` - Análise de presença online
- `/admin` - Área do proprietário
- `/login` - Login

### 🛠 Tecnologias

- HTML5 + CSS3
- Vanilla JavaScript (ES6+)
- n8n (Automação)
- Twilio (WhatsApp API)
- Sheety (Database)
