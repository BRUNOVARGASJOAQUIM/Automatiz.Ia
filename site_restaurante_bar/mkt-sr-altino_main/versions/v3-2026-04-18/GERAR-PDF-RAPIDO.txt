## 💡 DICAS

### PDF sai com logo distorcido?


### Letras muito pequenas?
```

```

### Fundo branco saindo cinza?
```
Chrome: Marcar "Imprimir fundo" (opcional)
Firefox: Marcar "Imprimir fundo"
```

### Salvar em local específico?
```
Ctrl+P → Salvar → Escolher pasta
```
